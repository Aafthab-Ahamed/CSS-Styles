# CSS Pattern Gallery

A collection of procedural textures and patterns created entirely using CSS `linear-gradient` and `radial-gradient` functions. No external image assets are required for the backgrounds.

## 🚀 Features

* **Pure CSS:** Zero external dependencies or image files.
* **Responsive Design:** Adapts to different screen sizes using CSS Media Queries and flexible units (`vw`, `rem`).
* **Performance:** Faster load times compared to bitmap-based patterns.

## 🎨 Pattern Overview

1.  **Section One:** A grid-based blue checked pattern.
2.  **Section Two:** A complex geometric "diamond" weave using multi-layered gradients.
3.  **Section Three:** A technical "blueprint" or "HUD" style with dots and teal lines.
4.  **Section Four:** A minimalist dot-matrix pattern on a seagreen background.
5.  **Section Five:** An interlocking "scales" or "circles" pattern.

## 🛠️ Installation & Usage

1.  Clone or download this repository.
2.  Open `index.html` in any modern web browser.
3.  To use a pattern in your own project, simply copy the `background-image`, `background-size`, and `background-position` properties from the desired class in `style.css`.

## 📱 Responsiveness

The project uses a mobile-first mindset:
* **Desktop:** Large display cards with 40rem max-width.
* **Mobile:** Cards shrink to 90% of the viewport width and adjust height for better scrolling ergonomics.
